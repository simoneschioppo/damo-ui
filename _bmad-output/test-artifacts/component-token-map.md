## Mapping Notes

- Direct `var(--X)` references appear as **var refs**.
- Tailwind utilities (`bg-primary`, `text-foreground`, etc.) are flagged as **utility** usage.
- Per-instance overrides (`[--memphis-shadow-color:var(--primary)]`) appear as **override**.
- Derivative utilities (e.g., `bg-primary/90`) still flag the base token (`--primary`).
- For polymorphic/wrapper components, only the wrapper's own styles are listed.
- Memphis identity tokens (`--memphis-border-color`, `--memphis-shadow-color`) are explicit overrides or direct references.
- Z-index tokens ensure proper stacking: `z-dropdown` < `z-header` < `z-overlay` < `z-modal` < `z-toast` < `z-tooltip`.
- Motion tokens (`duration-*`, `ease-*`) control animation timing across all interactive components.
- All 54 components analyzed; 50 use custom properties, 4 use only primitive HTML styling.
