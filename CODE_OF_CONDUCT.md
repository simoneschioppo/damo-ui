# Code of Conduct

This project follows the **Contributor Covenant**, version 2.1. The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Summary

Everyone interacting in `damo-ui` spaces — issues, pull requests, discussions, and any other project channel — is expected to follow the standards outlined in the Contributor Covenant linked above. The short version: be respectful, assume good faith, focus on the technical work, and remember that there is another human on the other side of the keyboard.

## Reporting

If you experience or witness behavior that violates the Code of Conduct, please report it privately to the maintainer:

- **GitHub:** open a private security advisory at <https://github.com/simoneschioppo/damo-ui/security/advisories/new> and mark the report as a Code of Conduct concern.

All reports will be reviewed and handled with discretion. The maintainer is committed to protecting the privacy and safety of anyone who reports an issue in good faith.

## Enforcement

Consequences for violations follow the Contributor Covenant enforcement ladder (correction, warning, temporary ban, permanent ban). The full ladder is documented in the linked Covenant text.
