# core-knowledge/

This folder is **output-only**. Do not hand-edit files here.

All content is produced by the **Kipi** agent (`_bmad/agents/kipi/kipi.md`)
through one of its five workflows. Hand edits break the audit trail
(`Last scan: <sha>` becomes a lie) and will be flagged as outdated on the
next `*5 Status` run.

## How to write here

Activate Kipi and run a workflow:

| You want to…                                          | Run            |
|-------------------------------------------------------|----------------|
| Bootstrap the chapter layout                          | `*1` Initialize |
| Document a piece of code                              | `*2` Scan      |
| Process notes/specs from `user-data/`                 | `*3` User Data |
| Sync chapters with recent code or new processed notes | `*4` Update    |
| See what's documented / pending / outdated            | `*5` Status    |

## Layout

The structure is decided by `*1 Repository Initialize` on first run. Until
then, this folder contains only `ready-for-classify/` (a staging area for
processed user-data drops awaiting merge into chapters).

## Language

Conversation with Kipi can be in Italian. Files in this folder are written
in **English** for long-term portability.
